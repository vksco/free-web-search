# Attribution & Credits

## What This Skill Is

This is a **wrapper/integration skill** that makes it easy to use [SearXNG](https://github.com/searxng/searxng) with OpenClaw AI assistants.

## What This Skill Is NOT

- ❌ This is NOT a search engine itself
- ❌ This is NOT a reimplementation of search functionality
- ❌ This is NOT competing with SearXNG

## What This Skill DOES

- ✅ Provides easy Docker setup for SearXNG
- ✅ Integrates SearXNG with OpenClaw
- ✅ Adds auto-update functionality
- ✅ Provides documentation for AI agent usage
- ✅ Adds Python helper functions

```
User Query
    ↓
This Skill (Wrapper)
    ↓
SearXNG (Core Engine)
    ↓
70+ Search Engines (Google, Bing, etc.)
    ↓
Aggregated Results
    ↓
Back to User
```

**This skill** = Wrapper that manages Docker and provides helper functions
